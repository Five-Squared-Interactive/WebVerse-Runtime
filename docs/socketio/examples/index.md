# Socket.IO Example World Scripts

Working examples for common Socket.IO patterns in WebVerse world scripts. Each example is self-contained and can be copied directly into your world script.

## Examples

- **[Basic Chat](basic-chat.md)** — Connect to a server, send and receive messages
- **[Room-Based Messaging](room-messaging.md)** — Join and leave rooms for scoped messaging
- **[Binary Transfer](binary-transfer.md)** — Send and receive binary data using base64 encoding
- **[Reliable Messaging](reliable-messaging.md)** — Use EmitWithAck for confirmed delivery of critical operations
- **[Connection Resilience](connection-resilience.md)** — Handle disconnects, reconnection, and connection failures with UI feedback

## Before You Start

All examples assume you have a Socket.IO server running. The examples use `https://my-server.com` as a placeholder — replace it with your server's URL.

For API details, see the [API Reference](../api-reference.md). For troubleshooting, see the [Error Handling Guide](../error-handling.md).
